### Frontend

1. Clone this repo
```bash
git clone https://github.com/luizaugustoliveira/Store.git 
```

2. Go to the project folder
```bash
cd Store
cd vtexstore
```

3. Install dependencies
```bash
npm install
```

3. Run Frontend
```bash
npm run dev
```

### Running the API


2. Go to the API folder

```bash
cd Store
cd api
```

3. Install dependencies
```bash
npm install
```

3. Run start
```bash
npm start
```
